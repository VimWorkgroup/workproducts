|----------------------|-----------------------------------------------|----------------------------------------|
| first line           |                                               | `vim9script`                           |
| comment sign         | `"`                                           | `#`                                    |
| variable declaration | `let MyVar=0`                                 | `var MyVar=0`                          |
| variable assignment  | `let MyVar=0`                                 | `MyVar=0`                              |
| function definition  | <code>function MyFunc()<br>endfunction</code> | <code>def MyFunc()<br>enddef</code>    |
